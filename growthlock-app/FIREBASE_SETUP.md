# 🔐 Firebase Authentication Setup Guide

## Overview
This GrowthLock app uses **Firebase Authentication** for secure password management and password reset functionality.

---

## 📋 Prerequisites

- Google account
- Firebase project (free tier available)
- Admin access to Firebase Console

---

## 🚀 Step-by-Step Setup

### **Step 1: Create a Firebase Project**

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Click **"Create a project"**
3. Enter project name: `growthlock-app`
4. **Disable** "Enable Google Analytics" (optional)
5. Click **"Create project"** and wait for completion

---

### **Step 2: Enable Email/Password Authentication**

1. In Firebase Console, go to **Authentication** (left panel)
2. Click **"Get started"**
3. In the **"Sign-in method"** tab, click **Email/Password**
4. Toggle **"Enable"** → **"Enable"**
5. Click **"Save"**

---

### **Step 3: Enable Password Reset Feature**

1. Go to **Authentication** → **Templates** (tab at top)
2. Click on **"Password reset"** email template
3. Set **Sender name**: `GrowthLock`
4. Keep default template (or customize email content)
5. Click **"Save"**

---

### **Step 4: Get Your Firebase Configuration**

1. Go to **Project settings** (click gear icon at top)
2. Under **"Your apps"**, click **"Web"** (🌐 icon)
3. Copy the config object that looks like:

```javascript
const firebaseConfig = {
    apiKey: "YOUR_API_KEY_HERE",
    authDomain: "growthlock-app.firebaseapp.com",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_STORAGE_BUCKET",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};
```

---

### **Step 5: Update Firebase Config in Your App**

1. Open **`forgot-password.html`** in your code editor
2. Find this section:

```javascript
const firebaseConfig = {
    apiKey: "AIzaSyDz_placeholder_YOUR_API_KEY",
    authDomain: "growthlock-app.firebaseapp.com",
    projectId: "growthlock-app",
    storageBucket: "growthlock-app.appspot.com",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};
```

3. **Replace** with your actual Firebase config values

---

### **Step 6: Configure Email Sender (Optional but Recommended)**

To send password reset emails from your custom email:

1. Go to **Authentication** → **Templates** → **Password reset**
2. Click **"Customize"** if you want custom email template
3. Message will say: *"Click the link below to reset your password"*
4. Users will receive email from Firebase (or your domain if custom configured)

---

### **Step 7: Set Up Authorized Domains**

1. Go to **Authentication** → **Settings** (not "Authorized domains" tab)
2. Scroll down to **"Authorized domains"**
3. Add your domains:
   - `localhost:8000` (for local testing)
   - `172.20.10.2:8000` (for phone testing)
   - `your-production-domain.com` (when deployed)
4. Click **"Add domain"** for each

## 🔧 How Password Reset Works

### **User Flow:**

1. ✅ User clicks **"Forgot password?"**
2. ✅ User enters their email on `/forgot-password.html`
3. ✅ Firebase sends secure reset email (expires in 1 hour)
4. ✅ User clicks link in email
5. ✅ Firebase password reset page loads (auto-generated by Firebase)
6. ✅ User enters new password
7. ✅ Password updated in Firebase Authentication
8. ✅ User redirected to login
9. ✅ User logs in with new password

---

## 🛡️ Security Features

✅ **Firebase handles:** 
- Secure password hashing
- Reset link expiration (1 hour)
- SSL/TLS encryption
- Email verification
- Brute-force protection

✅ **No passwords in database:**
- Passwords stored in Firebase Auth only
- Never visible in Realtime Database

✅ **Mobile-friendly:**
- Works on all devices
- Responsive email design

## 🐛 Troubleshooting

### **"Firebase is not defined"**
- Check that Firebase CDN script is loaded
- Verify internet connection
- Clear browser cache

### **"Email not found" error**
- User must be registered first
- Check email spelling
- Verify user exists in Firebase Authentication

### **Not receiving reset email**
- Check spam/promotions folder
- Verify email is correct
- Check Firebase email templates are configured
- Wait a few seconds (sometimes delayed)

### **"auth/network-request-failed"**
- Check internet connection
- Verify Firebase config is correct
- Check firewall settings

### **Reset link expired**
- Link expires after 1 hour
- User should request new reset email
